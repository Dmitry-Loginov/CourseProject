Для снятия ограничения на чтение с базы данных, нужно :
1. Открыть каталог с установленной программой, затем открыть папку DataBase и в ней выбрать файл Log.mdf
3. После этого нужно с вызвать контекстное меню, и выбрать пункт Свойства.
4. После этого следует перейти на вкладку Безопасность, и нажать кнопку Изменить
5. После этого в появившемся диалоговом окне для каждой группы или пользователя следует с помощью переключателя разрешить полный доступ
6. Затем нужно нажать кнопку Применить и ОК.
7. После этого контекстное меню можно закрывать
